Условные обозначения

[название_клавиши]

нажать указанную клавишу

[ Click ]

нажать и отпустить левую кнопку мыши

[ RMB ]

нажать и удерживать правую кнопку мыши

[ MMB ]

нажать и удерживать среднюю кнопку (колесо) мыши

[ Wheel ]

вращать колесо мыши

Управление камерой

— с помощью клавиатуры:

[вверх]

сдвинуть камеру вперед

[вниз]

сдвинуть камеру назад

[влево]

сдвинуть камеру влево

[вправо]

сдвинуть камеру вправо

[ Page Up ]

приблизить камеру

[ Page Down ]

отодвинуть камеру

[ Ctrl ] + [влево]

вращать камеру влево

[ Ctrl ] + [вправо]

вращать камеру вправо

[ Ctrl ] + [ вверх ]

наклонить камеру вверх

[ Ctrl ] + [вниз]

наклонить камеру вниз

[ Home ]

центрировать камеру на выбранного персонажа

— с помощью мыши:

[ Wheel ]

изменить масштаб (приблизить/отодвинуть камеру)

сдвиг влево + [ RMB ]

вращать камеру влево

сдвиг вправо + [ RMB ]

вращать камеру вправо

сдвиг вверх + [ RMB ]

наклонить камеру вверх

сдвиг вниз + [ RMB ]

наклонить камеру вниз

сдвиг + [ RMB ] + [ Ctrl ]

перемещать камеру в направлении сдвига мыши

сдвиг + [ MMB ]

перемещать камеру в направлении сдвига мыши

Отображение объектов

[ Num _+]

показать уровень (этаж) выше

[ Num _–]

показать уровень (этаж) ниже

[ Alt ]

вывести названия лежащих предметов,
вывести стоимость предметов у текущего персонажа

Выбор персонажей

[1]...[7]

выбрать персонажа по его номеру

[=]

выбрать всех персонажей

[ Click ] по закладке персонажа

выбрать указанного персонажа

[Tab], [Shift] + [Tab]

выбрать следующего персонажа (по кругу)

[ Ctrl ] + [ Click ] по закладке персонажа

добавить указанного персонажа в группу,

удалить указанного персонажа из группы

Управление персонажем

Передвижение:

[ Click ] по местности

указать конечную точку (пошаговый режим),

двигаться к точке (режим реального времени)

двойной [ Click ]

двигаться к точке (пошаговый режим),

бежать к точке (режим реального времени)

[Ctrl] + [Click]

двигаться, не обращая внимания на предметы под курсором

поза и действия:

[ M ]

двигаться (продолжить движение)

[ S ]

остановиться

[ A ]

атаковать

[ L ]

смотреть

[ Z ]

встать / бежать

[ X ]

встать / идти

[ C ]

присесть / красться

[ V ]

лечь / ползти

[ D ]

держать ориентацию

отображение окон:

[ I ]

вывести / скрыть окно предметов

[ U ]

вывести / скрыть окно параметров персонажа

стрельба (доступные режимы зависят от вида оружия):

[ Click ] по цели

выстрелить или начать снайперское прицеливание

[ R ]

перезарядить оружие

[ Q ]

выстрел навскидку ( snap shot )

[ W ]

прицельный выстрел ( aimed shot )

[ E ]

меткий выстрел ( careful shot )

[ [ ]

переключиться в предыдущий режим

[ ] ]

переключиться в следующий режим

[ F

короткая очередь

[G]

длинная очередь

[T]

режим снайперской стрельбы

диалоги:

[Пробел]

перейти к следующей фразе

[ Esc ]

пропустить все фразы и завершить диалог

Управление игрой

[ Enter ]

конец хода / перейти в пошаговый режим

[ F 5]

быстрая запись игры (если разрешено по сложности)

[ F 6]

[ F 8]

[ Pause ]

[Print Screen]

[ Scroll Lock ]

вывести / скрыть статистику ( fps и т.д.)

[ Ctrl ] + [ Q ]

выйти из игры в Windows
